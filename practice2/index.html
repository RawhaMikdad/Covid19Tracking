About

EnvidStat = Environment + Covid + Statistics MAKE INFORMED DECISIONS OF GOING OUTSIDE.
EnvidStat is a webapp which uses real time weather data (Temperature and Wind speed) to predict vulnerability of corona attack in a particular location and visualize those datas. With this now you get to know whether it's safe for you to go outdoors and helps you to make informed decision.

EnvidStat Breaks Like

Environment

Envidstat being an webapp there is no need for downloads, hence, no storage required.It is a responsive webapp, which means it adapts to any device you use to acccess it.

Covid-Update

EnvidStat is "Intuitive" Envidstat is full of data visualizations making it easier for you to process all the information of Covid-19 areound you.

Statistic

EnvidStat is Providing probable Statictical report of Covid-19 based on the real-time Environment.

Web-App
